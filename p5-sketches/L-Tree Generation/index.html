L-Tree Generation

An assortment of different context-free languages that, using turtle graphics axioms, create different effects.
Pressing the buttons you can see the different steps in their creation (or by pressing "A") and also apply a slight random effect
to them by using the "Disperse" button(or by pressing "S") ( works best on plant inspired axioms). Sadly, in this view, not all
axioms work as intended. Find more about this project here.
